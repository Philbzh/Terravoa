# Terravoa Marketplace UI Kit

Click-thru recreation of the Terravoa marketplace web app. Faithful to `MyTerraVera/src/components/`.

## Components
- `Navbar.jsx` — Sticky three-column nav (links · wordmark · account/cart)
- `Footer.jsx` — Four-column footer + newsletter line
- `Hero.jsx` — Full-bleed image hero with green protection gradient
- `EuropeStrip.jsx` — 8-region origin grid
- `RegionCards.jsx` — Region archive tiles
- `ProductCard.jsx` — Square image · meta line with flag · serif title · price · ships line
- `StoryOfTheWeek.jsx` — Asymmetric 2-col with rotated portrait
- `ProducerCTA.jsx` — Forest-green rounded CTA card
- `ProductDetail.jsx` — PDP screen
- `ProducerProfile.jsx` — Producer profile screen
- `Collection.jsx` — Filtered product grid

## Screens (click-thru via top tab bar in index.html)
- Homepage
- Collection
- Product detail
- Producer profile

JSX is loaded via Babel; React 18.
